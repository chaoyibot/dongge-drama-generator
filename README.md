# 东哥短剧生成器

一个基于 Agnes AI 的 Web 短剧生成工具，支持从剧本到成片的完整流程。

## 功能特点

- 📝 智能剧本生成
- 🎭 角色设定图生成
- 🎨 场景参考图生成
- 🎥 视频片段自动生成
- 🎞️ 成片自动拼接

## 使用方法

直接在浏览器中打开 `index.html` 文件即可使用。

## 技术栈

- 前端：原生 HTML/CSS/JavaScript
- AI 服务：Agnes AI
- 视频处理：FFmpeg
